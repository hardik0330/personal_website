My Projects

These here are some of my projects that were developed by me during my college period. Most of my projects are based on robotics, electronics as well as Computer vision and web Development.

Talking Network

Talking Network is a web application that has the capabilities of displaying water network elements along with a Voice
User Interface.

Wall Following Maze Solver Robot

This robot was developed for the E-yantra competition.
I was provided with the e-YFi Mega board developed by
IIT Bombay, this board works on the AVR platform.

Wireless Robot controlled by python script
Wireless Transmission mode - Radio Frequency
(NRF24L01)

This project is a basic demonstration of wireless communication between two microcontrollers via NRF24L01 modules. Continuous communication takes place between the two microcontrollers.
The user input is provided by keyboard of a computer, which is then sent to arduino connected to the computer via pyserial library.

Wireless Robot driven by RF Modules

Spider Robot

It is a nature inspired robot having 4 legs, this robot is inspired
by the movement of a spider. I used Servo motors to make the
legs of the robot, two servo motors were used in making
one leg of the robot.
